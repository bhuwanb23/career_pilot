# Phase 3.5 — Tool Registry

## Goal
Register every capability as a discoverable tool. AI can query available tools, their inputs/outputs, and invoke them.

---

## Current State
- 9 intent handlers exist as workflows
- Services are scattered across `services/*.py`
- No unified tool abstraction
- No way for the AI to discover what tools are available
- MinerU doesn't exist in the codebase (PyMuPDF used for PDF extraction)

---

## Design: Tool Registry System

### Tool Base Class

Each tool exposes:
- `name` — unique identifier
- `description` — what it does (for AI discovery)
- `category` — grouping (Resume, JD, CareerOps, MinerU, Interview, Database)
- `input_schema` — JSON Schema describing expected inputs
- `output_schema` — JSON Schema describing outputs
- `execute(**kwargs)` — async function that runs the tool

### Tool Categories

| Category | Tools |
|----------|-------|
| **Resume** | `resume_parse`, `resume_generate` |
| **JD** | `job_analyze` |
| **CareerOps** | `cover_letter_generate`, `recruiter_msg_generate`, `profile_get`, `applications_list` |
| **MinerU** | `document_extract` (PDF/DOCX → structured text) |
| **Interview** | `interview_prep` |
| **Database** | `analytics_get` |

## Step 1: Create Tool Base Class and Registry

**New file**: `backend/services/tool_registry.py`

```python
from dataclasses import dataclass, field
from typing import Any, Callable, Awaitable

@dataclass
class Tool:
    name: str
    description: str
    category: str
    input_schema: dict   # JSON Schema
    output_schema: dict  # JSON Schema
    execute: Callable[..., Awaitable[Any]]

class ToolRegistry:
    def __init__(self):
        self._tools: dict[str, Tool] = {}

    def register(self, tool: Tool):
        self._tools[tool.name] = tool

    def get(self, name: str) -> Tool | None:
        return self._tools.get(name)

    def list_all(self) -> list[dict]:
        return [{"name": t.name, "description": t.description, "category": t.category,
                 "input_schema": t.input_schema, "output_schema": t.output_schema}
                for t in self._tools.values()]

    def list_by_category(self, category: str) -> list[dict]:
        return [t for t in self.list_all() if t["category"] == category]

    def to_system_prompt(self) -> str:
        """Generate a system prompt snippet listing all available tools."""
        lines = ["Available tools:"]
        for cat in sorted(set(t.category for t in self._tools.values())):
            lines.append(f"\n[{cat}]")
            for t in self._tools.values():
                if t.category == cat:
                    lines.append(f"  - {t.name}: {t.description}")
                    lines.append(f"    Input: {json.dumps(t.input_schema)}")
        return "\n".join(lines)

# Singleton
registry = ToolRegistry()
```

## Step 2: Register Existing Services as Tools

**New file**: `backend/services/tools/__init__.py`

Registers all 11 tools by wrapping existing service functions:

### Resume Tools

**`resume_parse`** — wraps `resume_parser.parse_resume()`
- Input: `{ "file_content": "base64 encoded PDF bytes" }`
- Output: `{ "raw_resume": str, "summary": str, "skills": list, "projects": list, "education": list, "experience": list }`

**`resume_generate`** — wraps `resume_generator.generate_resume()` + `resume_to_pdf()`
- Input: `{ "profile_data": dict, "job_description": str (optional) }`
- Output: `{ "resume_data": dict, "pdf_base64": str }`

### JD Tools

**`job_analyze`** — wraps `job_analyzer.analyze_job()`
- Input: `{ "job_description": str, "profile_data": dict }`
- Output: `{ "company": str, "role": str, "match_score": float, "match_analysis": str, "cover_letter": str, "recruiter_msg": str }`

### CareerOps Tools

**`cover_letter_generate`** — wraps `cover_letter.generate_cover_letter()`
- Input: `{ "profile_data": dict, "company": str, "role": str, "job_description": str, "tone": str }`
- Output: `{ "cover_letter": str }`

**`recruiter_msg_generate`** — wraps `recruiter_msg.generate_recruiter_msg()`
- Input: `{ "profile_data": dict, "company": str, "role": str, "channel": str }`
- Output: `{ "message": str }`

**`profile_get`** — wraps `profile_service.get_profile()` + `profile_to_dict()`
- Input: `{ "db": Session }` (internal)
- Output: `{ "summary": str, "skills": list, "projects": list, "education": list, "experience": list }`

**`applications_list`** — wraps DB query
- Input: `{ "status": str (optional) }`
- Output: `{ "applications": list, "total": int, "by_status": dict }`

### MinerU Tools

**`document_extract`** — wraps MinerU (with PyMuPDF fallback)
- Input: `{ "file_content": "base64 PDF bytes", "format": "pdf" }`
- Output: `{ "text": str, "pages": int, "metadata": dict }`
- Implementation: Try `mineru` package first, fallback to `fitz` (PyMuPDF)

### Interview Tools

**`interview_prep`** — wraps `interview_prep.generate_prep()`
- Input: `{ "company": str, "role": str, "job_description": str, "profile_data": dict }`
- Output: `{ "company_summary": str, "questions": list, "star_answers": list }`

### Database Tools

**`analytics_get`** — wraps `analytics.get_raw_analytics()`
- Input: `{}` (no params needed)
- Output: `{ "total_applications": int, "status_breakdown": dict, "avg_match_score": float, ... }`

async def extract_document(file_content: bytes, filename: str = "") -> dict:
    """Extract text from document. Tries MinerU first, falls back to PyMuPDF."""

def _extract_with_pymupdf(file_content: bytes) -> dict:
    import fitz
    doc = fitz.open(stream=file_content, filetype="pdf")
    text = "".join(page.get_text() for page in doc)
    pages = len(doc)
    doc.close()
    return {"text": text.strip(), "pages": pages, "metadata": {}, "engine": "pymupdf"}

## Step 4: Expose Tools via REST API

**New file**: `backend/routers/tools.py`

```python
@router.get("/api/tools")
def list_tools():
    """List all available tools with their schemas."""
    return registry.list_all()

@router.get("/api/tools/{tool_name}")
def get_tool(tool_name: str):
    """Get details of a specific tool."""
    tool = registry.get(tool_name)
    if not tool:
        raise HTTPException(404, "Tool not found")
    return {...}

@router.post("/api/tools/{tool_name}/execute")
async def execute_tool(tool_name: str, body: dict, db: Session = Depends(get_db)):
    """Execute a tool by name."""
    tool = registry.get(tool_name)
    if not tool:
        raise HTTPException(404, "Tool not found")
    result = await tool.execute(db=db, **body)
    return result
```

# Design Patterns Summary

## Color System (Figma tokens in index.css)
- Primary: #000000 (black)
- Canvas: #ffffff
- Surface: #f7f7f5
- Hairline: #e6e6e6
- Pastel blocks: lime, lilac, cream, pink, mint, coral, navy
- Brand: indigo (#6366f1)

## Layout Patterns
- AppLayout: h-screen, flex-col, 3-panel (sidebar + main + right sidebar)
- Sidebar: w-16 collapsed, w-60 expanded
- RightSidebar: w-16 collapsed, w-72 expanded
- Main: flex-1, overflow-y-auto

## Component Patterns
- Cards: bg-white rounded-2xl border border-figma-hairline
- Buttons: pill-shaped (rounded-full), black primary, white secondary
- Transitions: transition-all-smooth, hover-lift, press-scale

## Pages Created
- Dashboard: Stats, charts, activity, tasks
- Workspace: AI agent chat
- Job Analysis: JD input, match score, skills
- Applications: Kanban board with 6 columns
- Interview Hub: Company summary, questions, STAR answers
- Pipeline: Job list with timeline
- Approved: Offer management
- Profile: Career profile display
